# Spring Data JDBC Basics

Spring Data JDBC offers a repository abstraction based on JDBC.

- The central interface in the Spring Data repository abstraction is Repository

```
/h2
jdbc:h2:mem:testdb
```

- [ ] Dependencies
- [ ] JDBCTemplate

- [ ] Repositories
- [ ] Data Relationships: 1:1, 1:n, n:m

- [ ] Transactions  
- [ ] Loading Schema
- [ ] Loading Data
- [ ] Testing
